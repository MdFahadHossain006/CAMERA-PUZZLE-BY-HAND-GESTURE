 A futuristic, glassmorphic puzzle game that uses **real-time camera input**, **face detection**, and **hand tracking** to create an interactive sliding puzzle experience.
 The game is a modern Glassmorphism UI to transform your live camera snapshot into a playable sliding puzzle.

### Features

- 🎥 Real-Time Camera Feed
- 🤖 AI Face Detection
- ✋ MediaPipe Hand Tracking
- 📷 Automatic Snapshot Capture
- 🧩 3×3 Sliding Puzzle
- 👆 Finger Gesture Tile Control
- ⏱ Live Timer
- 🔢 Move Counter
- 🎯 Completion Percentage
- 📈 Accuracy Calculation
- 🏆 Best Score Tracking
- 💎 Premium Glassmorphism UI
- 📱 Responsive Design
- ⚡ Smooth Animations
- 🔇 Sound Toggle

## 🚀 How It Works

1. Start the camera.
2. Align your face inside the frame.
3. Hold still for 3 seconds.
4. The app automatically captures your photo.
5. Your image becomes a 3×3 sliding puzzle.
6. Use your index finger to move puzzle tiles.
7. Complete the puzzle with the fewest moves and best accuracy.
